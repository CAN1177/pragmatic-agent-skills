# Roadmap

## Near Term

- add one more high-signal skill with bundled script support
- add installation notes for common agent runtimes
- add a lightweight validation checklist for every skill
- standardize naming conventions across skill directories

## Medium Term

- publish a reusable skill authoring template
- add example fixtures for input and expected behavior
- add CI checks for broken links and missing required skill files

## Longer Term

- make it easier to install a skill directly into common agent environments
- document a recommended test pattern for skills with local scripts
- expand the repository into a stronger reference for agent skill packaging
